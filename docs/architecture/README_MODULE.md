### `POST /v1/{{module}}/ingest`
**Description:** Ingest payload into module.

**Request (application/json):**
```json
{
  "provider": "example_provider",
  "timestamp": "2026-03-22T12:00:00Z",
  "records": [
    {"symbol": "AAPL", "price": 183.45, "volume": 1200}
  ]
}
```

**Response (200 OK):**
```json
{
  "status": "accepted",
  "job_run_id": "{{job_run_id}}",
  "ingested": 1
}
```

**Curl Example**
```bash
curl -X POST https://localhost:8000/v1/{{module}}/ingest \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer ${API_TOKEN}" \
  -d '{"provider":"example_provider","timestamp":"2026-03-22T12:00:00Z","records":[{"symbol":"AAPL","price":183.45,"volume":1200}]}'
```
